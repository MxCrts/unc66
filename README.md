# UNC 66 — Site vitrine

Site de l'UNC 66 (Union Nationale des Combattants, Pyrénées-Orientales), en
React + Vite + Tailwind CSS + Firebase.

Voir **[CLAUDE.md](./CLAUDE.md)** pour la charte graphique complète, l'état
d'avancement et les conventions du projet.

## Démarrer

```bash
npm install
npm run dev
```

## Build de production

```bash
npm run build
```
